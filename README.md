# 🔥 FOUL Lang - A Custom Programming Language

Welcome to **FOUL Lang** — a custom-built programming language designed as part of a personal learning project. This language is a work-in-progress and aims to explore how compilers, interpreters, and language design work under the hood.

## 🚀 Features (Planned & Implemented)

- [x] Basic syntax parser
- [x] Tokenizer / Lexer
- [x] Custom grammar rules
- [ ] Expression evaluation
- [ ] Variable declaration and assignment
- [ ] Control flow: if/else, loops
- [ ] Function definitions
- [ ] Error handling
- [ ] REPL (Read-Eval-Print Loop)
